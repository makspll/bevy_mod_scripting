# Scenario DSL

![alt text](https://github.com/makspll/bevy_mod_scripting/tree/main/scenario_file_language_server/image.png)

Syntax highlighting and completions for Scenario DSL files (`.scenario`) from the [bevy_mod_scripting](https://github.com/makspll/bevy_mod_scripting) framework based on a JSON schema.


# Developing & Usage

Run:
```bash
npm install
npx tsc
npx vsce package
```

Then install into vscode via generated `.vsix` file
